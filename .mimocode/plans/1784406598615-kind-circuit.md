# Fix: Remove "Hi" Welcome page, redirect `/` to dashboard

## Problem
After login, going to `/` shows a "Hi" page. Should redirect to `/dashboard`.

## Changes

### 1. Delete `resources/js/pages/Welcome.vue`
Remove the file entirely.

### 2. Update `routes/web.php` line ~25
Replace:
```php
Route::inertia('/', 'Welcome')->name('home');
```
With:
```php
Route::get('/', fn () => redirect()->route('dashboard'))->name('home');
```

### 3. Update `resources/js/app.ts` layout function
Remove the `case name === 'Welcome': return null;` case since Welcome.vue no longer exists.

## Verification
- `curl https://xiaomi-mimo.test/` should 302 to `/dashboard` (if logged in) or `/login` (if not)
- `php artisan test --compact` should still pass
