# Install EvoSkill in Project

## What is EvoSkill

EvoSkill is a Python-based CLI tool that automatically discovers and synthesizes reusable agent skills from failed trajectories to improve coding agent performance. It's compatible with Claude Code (which this project uses).

**It is NOT a Laravel/PHP package** — it's a standalone Python tool that runs alongside the project.

## Prerequisites

- Python 3.12+ (system has 3.9.6 — `uv` will handle downloading the correct version)
- `uv` (installed: v0.9.5)
- Git (installed)
- Claude Code CLI (installed)

## Installation Steps

### Step 1: Clone and install EvoSkill globally

```bash
# Clone the repo to a tools directory
git clone https://github.com/sentient-agi/EvoSkill.git ~/.local/share/EvoSkill

# Install dependencies using uv (handles Python 3.12+ automatically)
cd ~/.local/share/EvoSkill && uv sync

# Make evoskill CLI available globally
uv pip install -e .
```

### Step 2: Initialize EvoSkill in the project

```bash
cd /Users/andrejprus/Herd/xiaomi-mimo
evoskill init
```

Select during interactive setup:
- Agent runtime: **claude**
- Dataset source: **CSV**
- Path: **data/laravel_questions.csv**
- Question column: **question**
- Answer column: **answer**
- Execution mode: **Local**

This creates:
- `.evoskill/config.toml` — configuration file
- `.evoskill/task.md` — task description template

### Step 3: Configure `.evoskill/config.toml`

```toml
[harness]
name = "claude"
model = "sonnet"

[evolution]
mode = "skill_only"
iterations = 20
frontier_size = 3
concurrency = 4
no_improvement_limit = 5

[dataset]
path = "data/laravel_questions.csv"
question_column = "question"
answer_column = "answer"
train_ratio = 0.18
val_ratio = 0.12

[scorer]
type = "multi_tolerance"
```

### Step 4: Create Laravel dataset

Create `data/laravel_questions.csv` with Laravel coding questions relevant to this project:

```csv
question,answer
"How do you create a new Eloquent model in Laravel?","php artisan make:model ModelName"
"What is the command to run migrations?","php artisan migrate"
"How do you define a route in Laravel?","Route::get('/path', [Controller::class, 'method']);"
"How do you create a form request for validation?","php artisan make:request FormRequestName"
"What is the command to clear all caches?","php artisan optimize:clear"
...
```

### Step 5: Run the evolution loop

```bash
evoskill run
```

## Important Notes

1. **EvoSkill modifies git branches** — it creates `program/*` branches and `frontier/*` tags. Your `main` branch is untouched.
2. **API keys required** — needs `ANTHROPIC_API_KEY` for Claude Code harness
3. **Python 3.12+** — `uv` will auto-download the correct Python version if needed
4. **Results live in `.claude/skills/`** — evolved skills are placed there for Claude Code to use

## Files Created/Modified

| File | Purpose |
|------|---------|
| `.evoskill/config.toml` | EvoSkill configuration |
| `.evoskill/task.md` | Task description for evolution |
| `.claude/skills/` | Evolved skills (populated after `evoskill run`) |
| `.claude/program.yaml` | Best program config (populated after run) |
| `program/*` git branches | Version history of evolved programs |

## Verification

After installation:
```bash
evoskill --version    # Confirm CLI works
evoskill init         # Initialize in project
evoskill skills       # List discovered skills (empty initially)
```
