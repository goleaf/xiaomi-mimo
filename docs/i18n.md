# Internationalization

## Required Locales

The application must support English (`en`), Lithuanian (`lt`), and Russian (`ru`), with English fallback. Translation keys must be stable semantic identifiers grouped by domain.

## Formatting

Dates, times, numbers, relative time, calendars, and server-side boundaries must use the selected locale, timezone, date format, time format, and first day of week. Canonical timestamps remain storage values; locale affects presentation and boundary calculation.

## Current State

There is no complete translation architecture. User-facing PHP and Vue strings are largely hardcoded, enum labels are derived directly from stored values in places, and several components hardcode `en-US`. Prompt 7 must introduce one documented backend/frontend translation source, persistence, HTML language metadata, fallback handling, and key-parity tests.
